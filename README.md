# Planner GUI 🗓️

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

**Planner GUI** is a simple desktop application that helps you organize tasks and plan your day. Built with Python, it provides a clean and user-friendly interface for daily and weekly planning.  

---

## Features ✨
- Add, edit, and delete tasks  
- Daily and weekly planner view  
- Simple, intuitive GUI  

---

## Installation 🛠️

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/username/planner_gui.git
   cd planner_gui

   USAGE 📝

Open the app and start adding your tasks.

Edit or delete tasks as needed.

Close the app to save your progress.
